# VolariX Legal Requirements

## Mandatory Disclaimer
Every page and signal-generating component must display:
> "VolariX provides educational and analytical tools only. All signals, risk metrics, and data points are not financial advice. Trade at your own risk."

## Where It Must Appear
- Footer of index.html
- Every tab showing: signals, scores, IV data, options strategies, sector recommendations, congressional trades, insider trades, paper trade results
- Risk Shield modal, Scorecard modal, Profit Calculator modal, Settings page footer

## Rules
- Never promise returns or guaranteed outcomes
- Never use "will go up", "safe trade", "guaranteed profit"
- All analytics labelled as educational tools only
- Premium features do not imply financial advisory services
- Paper trading results explicitly labelled as simulated
