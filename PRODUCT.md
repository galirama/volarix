# VolariX Product Specification

## Product Vision
VolariX is a cross-platform fintech web application for risk-first options trading.

Primary goal:
Help retail traders reduce losses and improve discipline using analytics, alternative data, and simulated trading.

Core principle:
"Protect capital before maximizing gains."

---

# Target Users

1. Beginner options traders
2. Intermediate premium sellers
3. Swing traders
4. Risk-conscious retail investors

---

# MVP Scope (Phase 1)

## Free Features
- Secure login with MFA
- Responsive dashboard
- Stock ticker search
- Technical indicators:
  - Moving Average (MA)
  - MACD
  - RSI
- Fundamental metrics
- Bullish/Bearish signal generation
- Support & resistance levels
- Paper trading account
- Global ticker banner:
  - CNN Fear & Greed Index
  - VIX

## Premium Features
- High IV screener
- Greek Visualizer
- Congressional trading tracker
- Insider buying tracker
- VolariX Risk Shield
- AI Post-Mortem Log
- SMS alerts

---

# Monetization

## Free Tier
- Delayed data
- Limited technical analysis
- Limited paper trading

## Premium Tier
- Risk Shield access
- Advanced options analytics
- Alternative data
- AI insights
- Real-time SMS alerts

Subscription provider:
Stripe

---

# Phase Rollout

## Phase 1 (MVP)
Auth + dashboard + stock search + indicators + paper trading

## Phase 2
Options screener + premium subscriptions

## Phase 3
Risk Shield + AI Companion

## Phase 4
Broker integrations

---

# Legal Requirement
All analytical outputs must show disclaimer:

"VolariX provides educational and analytical tools only. Nothing displayed is financial advice. Trade at your own risk."
